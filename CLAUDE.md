# Проект: Warehouse Management App

## Назначение
Веб-приложение для учета склада кафе-джелатерии. Дополняет систему SABY, 
не заменяет её. Используется для еженедельных выборочных инвентаризаций 
по 4 складам: Бар, Лаборатория, Кухня, Основной склад.

## Стек
- Backend: Node.js 20 + Fastify + better-sqlite3 + JWT auth + bcrypt
- Frontend: React 18 + Vite + TailwindCSS + Zustand + React Router + PWA
- База: SQLite (файл database.sqlite)
- Деплой: VPS, PM2 для процесса, Nginx как reverse proxy

## Принципы
1. Soft delete везде (поле deleted_at, никакого DELETE FROM)
2. Audit log для каждого изменения данных
3. Автобэкапы каждые 6 часов + ежедневный в облако
4. Закрытые инвентаризации не редактируются, только корректирующие
5. Роли: admin (руководитель), manager, inventarizator
6. Все веса в килограммах (правило из SABY)
7. Основной склад всегда = 0 (транзитный, "золотое правило")

## Структура папок
/backend  — API сервер
/frontend — React приложение  
/docs     — документация и ER-диаграммы

## Стиль кода

- Комментарии на русском в бизнес-логике
